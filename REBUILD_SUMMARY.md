## üöÄ THE 13-STAGE WORKFLOW

### Stage 1: Event Creation
```
Input: Category, Gender, Event Name, Date, Time, Venue
Output: Event record + sample athletes generated
Status: statusFlow.created = true
```

### Stage 2: Call Room Generation
```
Generate printable sheet with all athletes
Format: SL NO | CHEST NO | NAME | COLLEGE | REMARKS | DIS
Print/PDF: ‚úÖ Supported
```

### Stage 3: Call Room Completion
```
Admin marks each athlete: PRESENT / ABSENT / DISQUALIFIED
Only PRESENT athletes proceed
Summary shows attendance counts
```

### Stage 4: Generate Event Sheets
```
Track: Sets of 8 athletes
Jump/Throw: Pages of 15 athletes
Relay: Teams of 4 (1 SL NO per team)
Combined: Total points entry sheet
```

### Stage 5: Round 1 Scoring
```
Track/Relay: Time entry (lower = better)
Jump/Throw: Distance entry (higher = better)
Combined: Total points entry (higher = better)
Auto-ranking based on category
```

### Stage 6: Select Top Athletes
```
Top 8 athletes selected from rankings
Eliminated athletes dropped
Ready for heats/finals
```

### Stage 7: Generate Heats
```
Track/Relay: Lane assignments from fixed pattern [3,4,2,5,6,1,7,8]
Heat 1: Odd ranks (1,3,5,7)
Heat 2: Even ranks (2,4,6,8)
Jump/Throw/Combined: No heats generated
```

### Stage 8: Pre-Final Sheet
```
Printable sheet with lanes assigned
Empty timing column for officials
Ready for finals
```

### Stage 9: Final Scoring
```
Admin enters final performances
May differ from Round 1
Re-ranking based on final scores
```

### Stage 10: Final Announcement
```
Medal winners displayed (ü•áü•àü•â)
Points awarded (5-3-1 system)
College totals calculated
```

### Stage 11: Name Correction
```
Verify & edit athlete details
Editable: Name, College, Chest Number
Before publication
```

### Stage 12: Verification
```
Checklist blocks incomplete events
All 7 stages must be checked:
- Call Room Completed
- Sheets Generated
- Round 1 Scored
- Heats Generated
- Pre-Final Generated
- Final Scored
- Name Correction Done
```

### Stage 13: Publish & Lock
```
Final warning: "This action CANNOT be undone"
Event locked permanently
No further editing allowed
statusFlow.published = true
statusFlow.lockedAt = timestamp
```
